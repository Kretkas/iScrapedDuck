# iPhone setup

The full Scriptable installation guide now lives in the main README:

- [Install on iPhone](../README.md#install-on-iphone)

Quick summary:

1. Create a new Scriptable script named `RaidHundoCP`.
2. Paste `scriptable/RaidHundoCP.js`.
3. Add a Scriptable widget to the Home Screen.
4. Edit the widget and select `RaidHundoCP`.

Use a Large widget if you want all raid rows visible at once.
